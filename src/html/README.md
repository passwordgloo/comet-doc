## 语法规范

### HTML的注释

#### 注释的含义

注释是方便程序员下次看到此处时方便想起功能和含义，便于维护

使用<kbd>Ctrl + /</kbd> 添加行注释

#### 注释的作用

为代码添加具有解释性、描述性的信息，帮助开发人员理解代码

浏览器执行代码会忽略掉所有注释

### HTML标签的构成

```html
<strong>文字加粗</strong>
//开始标签 包裹内容 结束标签
```

标签由`<` `>` `/`英文单词或字母组成

常见标签由两部分组成，称之为双标签

少数标签由一部分组成，称之为单标签

### HTML标签关系

1. 父子关系（嵌套关系）
2. 兄弟关系（并列关系）

### 文件

| 命令     | 快捷键                                        |
| -------- | --------------------------------------------- |
| 打开     | <kbd>Ctrl</kbd>+<kbd>O</kbd>                  |
| 新建     | <kbd>Ctrl</kbd>+<kbd>N</kbd>                  |
| 存储     | <kbd>Ctrl</kbd>+<kbd>S</kbd>                  |
| 存储为   | <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>S</kbd> |
| 存储副本 | <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>S</kbd>   |

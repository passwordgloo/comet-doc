## 发展历程
Thomas Knoll 最初只是出于爱好编写一个灰阶图片工具[^1]，在兄弟John Knoll说服下，逐步扩展更多功能，将这款个人工具打造成商业软件。最初软件的名字叫Display，后来一次展会上兄弟二人接受观众建议，将其改为Photoshop。

出于商业化考量，兄弟二人试图将软件卖给其他公司，却屡屡碰壁，后来遇见Adobe艺术总监Russell Brown，二人将Photoshop软件分发权卖给了Adobe公司，但保留Photoshop版权，当时Adobe公司也没想到PS会有今天的地位，就爽快的同意了。进入Adobe公司后，Thomas Knoll一直是Photoshop开发团队的核心，PS CS4版本后，他的工作重心转向Camera RAW

1990年，Photoshop 1.0在Mac上首发，直到版本2.0才支持Windows平台。此后经过多次迭代开发，先后推出CS系列和今天的CC(creative cloud)系列

不同于其他图片编辑工具，PS还可以绘制简易动画、编辑PDF文档、RAW编辑

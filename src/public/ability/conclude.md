> 归纳概括= 总括句+分条、分点

## 单一性归纳

> [!warning]
>
> 有限字数多写关键词，不要浪费时间在组织文字

### 概括问题分类

概括主要问题：除了对策不写，其他的都要写

概括问题的表现：只写表现，多用原文中的现象

概括种种问题

### 概括原因

1. 多角度概括原因——五位一体（政治、经济、文化、社会、生态）、思想观念、法律、技术、财政等
2. 多主体概括原因
3. 内外因概括原因

### 概括危害

从小到大、从当前到未来、从个人到集体、社会、国家、全人类

## 综合性概括

### 概括内容

| 类型   | 说明                                     |
| ------ | ---------------------------------------- |
| 调研类 | 侧重表现、原因、对策，如汇报材料、报告等 |
| 宣传类 | 宣传材料、宣传稿子                       |
| 方案类 | 主要写对策                               |
| 文头类 | 编者按、摘要、提纲等，很少考             |

# 社区概论

## 社区的含义和基本特征

### 社区的由来

::: info 资料链接

社区一词最早提出的是德国社会学家“[滕尼斯](https://wiki.cnki.com.cn/HotWord/1580439.htm)”

社区一词最早是[费孝通](https://wiki.cnki.com.cn/HotWord/15014.htm)等人在20世纪30年代翻译滕尼斯等人的著作而得来的

:::

::: details 社区的定义

社区是指聚居在一定地域范围内的人们所组成的~~社会生活~~共同体

:::

首先，在血缘、家庭、家族的自然基础上建立的*血缘*共同体

其次，逐步发展和分离为人们居住在一起的*地缘*共同体

最后，进一步发展为*精神*共同体

### 社区的基本特征

社区是社会的缩影

社区是人类活动的承载体

社区具有自己的文化特色

社区具有多种功能

社区是不断发展变化的

### 社区与其他组织的关系

#### 社区与社会

::: tabs

@tab 区别

社区的地域范围比社会小，更具体

社区成员比社会人员更具有凝聚力

社区功能比社会功能更加具体和专门化

@tab 联系

社区是社会的一部分，社区所拥有的人员、地域是社会的一部分

社区是是社会的缩影，社区内部人口与地域范围决定其内部关系的复杂程度

:::

#### 社区与社区社会组织

::: note 名词解释

社区社会组织是指以社会居民为主要成员，以满足社区居民的多样化需求为目的，由居民自发成立并自觉参与，以公益或共同利益为目的社会组织形态

:::

::: info 社区社会组织的作用

- 提供社区服务
- 广泛动员社区居民参与社区公共事务和公益事业
- 丰富群众性文化活动，提升社区居民民主生活品质
- 发挥社区社会组织在源头治理方面的积极作用，提升社区矛盾预防化解能力

:::

#### 社区与行政区划单位

行政区划单位属于政治学范畴，强调依靠行政力量、法定形成的，实现各种职能政权机构

## 社区的基本要素和类型

### 社区的基本要素

#### 人口要素

> 人口要素是社区的第一要素，社区人口一般包括：人口的数量、质量、构成和分布、流动状况

#### 地域要素

地域是社区居民从事社会生产和社会活动的场所，是人们基本的生存空间，是人文元素与环境因素结合的共同体

#### 组织要素

组织是有目的、有计划的满足居民生存和发展需要的各种团体和机构

正式组织，如社区党组织、居民委员会

非正式组织，如社区合唱团、舞蹈队

#### 文化要素

各具特色，社区相互区别的标志

#### 心理要素

::: info 关键词

`归属感`：人们对生活的地区会产生自豪或羞耻的情感

`认同感`：一个人与他人互动，把自己看成是社区的成员并承担一定的义务

:::

#### 物质要素

物质要素是衡量一个社区是否完整的重要尺度，也是社区发展的基础和前提

### 社区的类型

#### 依照经济、文化因素划分

|          | 农村                                                 | 城市                                                         |
| -------- | ---------------------------------------------------- | ------------------------------------------------------------ |
| 从事劳动 | 指居民以从事农业生产为主要谋生手段的区域社会。       | 城市社区指在特定的区域内，由从事各种非农业劳动的密集人口所组成的共同体 |
| 人口     | 人口密度低、同质性强，流动性小                       | 人口集中，异质性强                                           |
| 生活方式 | 风俗习惯和生活方式等受传统影响较大                   | 具有各种复杂的制度、信仰、语言和多样化的生活方式             |
| 组织结构 | 组织结构简单，职业分工不如城市复杂                   | 具有结构复杂的各种群体和组织                                 |
| 家庭作用 | 家庭在生活中起着重要作用，血缘关系重要，人际关系密切 | 家庭的规模和职能缩小，血缘关系淡化，人际关系松散             |
| 经济活动 | 经济活动简单                                         | 经济和其他活动频繁                                           |

#### 民政部门对城市社区的划分

- 单位型社区，家属楼

- 板块型社区，封闭

- 小区型社区

- 功能型社区，步行街


#### 以空间特征划分

- 法定社区

- 自然社区，自然形成的社区

- 专能社区[^1]，如中关村、华强北

- 虚拟社区

# 社区居民自治

## 社区居民自治的基本概念和要素

::: details 社区居民自治基本含义

社区居民自治是指社区居民有自我决定社区公共事务的权利以及权利行使的方式

社区居民自治是社区参与的最高层次与参与形式

:::

### 自治的基本要素

自治区域

自治组织，村委会和居委会

自治事权（自治的核心）

### 自治的特征

- 直接性
- 有限性
- 有序性

### 自治的方式

民主选举<span alt="black">前提</span>

民主决策<span alt="orange">核心</span>

民主管理<span alt="orange">核心</span>

民主监督<span alt="purple">民主监督</span>

::: info

民主选举是基础，民主决策是关键，民主管理是举措（根本），民主监督是保障

:::

### 社区居民委员会的性质和主要职责

自我管理、自我教育、自我服务的基层群众性自治组织。实行民主选举、民主决策、民主管理和民主监督

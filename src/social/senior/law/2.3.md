## 主体、对象和受众范围

### 主体

| 具体主体           | 职责                                                         |
| ------------------ | ------------------------------------------------------------ |
| 各级民政部门       | 负责统筹规划、组织实施、绩效评估                             |
| 各级财政部门       | 负责计划审核、经费安排、监督管理                             |
| 有关部门和群团组织 | 负责本系统、本行业社会工作服务需求评估、向同级民政部门申请社会工作服务计划并具体实施 |

### 对象

社会团体

民办非企业单位、基金会

企事业单位

### 受众范围

城市流动人口、农村留守人员

困难群体、受灾群众

特殊人群

## 程序和监督管理

::: tabs

@tab 程序

编制预算（民政部门）

组织购买（公开）

签定合同

指导实施

@tab 监督

1. 建立健全监管制度
2. 加强过程监管
3. 建立综合性评审机制（由购买方、服务对象及第三方组成）
4. 坚持多元评估（过程与结果、短期与长远、社会效益与经济效益结合）
5. 考评结果与后续购买服务挂钩
6. 建立征信管理制度

:::

                                    # 常识

                                    # 人文常识

                                    # 天干地支

                                    天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸

                                    地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、猪

                                    注

                                    十二生肖是十二地支的形象化代表

                                    # 科举考试

                                    考试合格者名次
                                    童试秀才
                                    乡试举人解元（第一名）
                                    会试贡士会元（第一名）
                                    殿试进士第一甲：状元、榜眼、探花，赐进士及第
                                    第二甲：传胪，赐进士出身
                                    第三甲：赐同进士出身

                                    # 行政机构与行政区划

                                    # 行政机构

                                    御史台：专职弹劾的监察机关

                                    太常寺：封建社会掌管礼乐的最高行政机关

                                    宋朝枢密院是军队的最高指挥机构，同时领导兵部的工作

                                    明清督察院是全国最高监察机构

                                    # 行政区划

                                    秦：郡——县

                                    汉：州——郡——县

                                    唐：道——州——县

                                    宋：州——县，路是直辖于中央并高于府、州、军、监

                                    # 棋牌

                                    中国围棋业余段分 7级，专业段分九级

                                    麻将起源于中国

                                    国际象棋起源于亚洲，后由阿拉伯人传入欧洲

                                    中国象棋起源于中国，棋子32个，黑红各16个

                                    # 文学

                                    # 专著

                                    “上善若水”出自老子的《道德经》

                                    “吾生也有涯，而知也有涯”出自《庄子》

                                    莫言中篇小说《红高粱》以山东省潍坊市高密东北乡为背景

                                    世界三大忏悔录，作者分别是奥古斯丁、卢梭、托尔斯泰

                                    # 哲学

                                    “我思故我在”是法国哲学家笛卡尔的哲学命题

                                    《人性论》是英国哲学家大卫·休谟的著作

                                    “他人即地狱”出自萨特《隔离审讯》，萨特是法国20世纪最重要的哲学家之一，代表作是《存在与虚无》

                                    # 公文

                                    ”誓“的内容偏重于出兵打仗前的盟誓

                                    “命”是指帝王针对具体事情发布的命令

                                    “奏”是古代大臣对政事有所陈述、批评、建议的陈述文书

                                    “章”是受封赠的大臣向皇帝谢恩的公文

                                    “诰命”是皇帝封赠官员的专用文书，五品以上授以诰命，六品以下授以敕命

                                    # 诗歌

                                    “黔首”指老百姓

                                    “南冠”泛指罪犯

                                    “青鸟”指信使

                                    “采薇”代指隐居生活

                                    # 建筑与壁画

                                    # 壁画

                                    《创世纪》壁画的作者是米开朗基罗

                                    # 建筑风格

                                    洛可可式建筑风格纤弱娇媚、华丽精巧

                                    泰姬陵是伊斯兰建筑的代表作

                                    哥特式建筑风格发源于12世纪的法国

                                    巴黎圣母院采用了哥特式建筑形式，早于法国大革命前建造

                                    # 音乐

                                    《十面埋伏》使用弹拨乐器——琵琶演奏

                                    《天鹅湖》古典芭蕾舞剧取材于民间传说

                                    # 足球

                                    “德劳内杯”是欧洲杯的奖项名称

                                    世界杯由全世界国家级球队参与，职业足球俱乐部没有参赛权利

                                    # 时间

                                    戌时（19-21）亥时（21-23）子时（23-1）丑时（1-3）寅时（3-5）

                                    知非：50；从心：70；鲐背：90；期颐：100

                                    # 历法

                                    农历属于阴阳历

                                    # 节气

                                    春雨惊春清谷天，夏满芒夏暑相连。秋处露秋寒霜降，冬雪雪冬小大寒

                                    # 节日

                                    “社日”，春社，指春季祭祀土地神的日子

                                    农历“三月三”是中国多个民族的传统节日，其中以壮族为典型

                                    # 运动

                                    举重、拳击按参赛者体重进行分级

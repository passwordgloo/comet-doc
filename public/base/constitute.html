                                          # 概述

                                          概念

                                          宪法是具有最高法律效力的国家根本法，它规定国家的根本制度和根本任务，保障公民基本权利，集中体现各种政治力量对比关系

                                          我国宪法历史沿革

                                          1908年《钦定宪法大纲》是中国历史上第一个宪法性文件

                                          1912年颁布的《中华民国临时约法》是我国唯一的一部资产阶级宪法性文件

                                          《共同纲领》在1954年第一部宪法颁布前起临时宪法作用

                                          我国现行宪法[1]于1982年颁布，历经1988、1993、1999、2004、2018[2]五次修正

                                          • 我国根本任务：集中力量进行社会主义现代化建设
                                          • 社会主义的本质
                                          • 社会主义的根本任务：解放生产力和发展生产力
                                          • 社会主义的根本目的：共同富裕
                                          • 国体（国家性质）是人民民主专政
                                          • 政体（政治组织形式）是人民代表大会制度

                                          # 宪法史

                                          世界上第一部资产阶级成文宪法：1787年美国宪法

                                          欧洲大陆第一部成文宪法：法国1791年宪法

                                          世界上第一部社会主义宪法：1918苏俄宪法

                                          # 宪法特征

                                          宪法的修改由全国人民大表大会常务委员会或者五分之一以上的全国人民代表大会代表提议，并由全国人民代表大会以全体代表的三分之二以上的多数通过，法律和其他议案由全国人民代表大会以全体代表的过半数通过

                                          # 基层群众自治组织

                                          村民委员会每届任期5年，可连选连任

                                          党的基层委员会每届任期一般为5年，党的总支部委员会和支部委员会每届任期一般为3年

                                          # 国家机构概述

                                          全国人大及其常委会的关系：隶属关系，常设机关

                                          国务院与地方政府：领导关系

                                          各级监察委员会：领导关系

                                          各级法院：监督关系

                                          县级及以上：人大领导下的政府+法院+检察院

                                          乡级：人大只领导乡政府，无法院、检察院

                                          任期与选举

                                          国家主席、副主席、中央军委主席任期同全国人民代表大会每届任期相同（5年）

                                          监察委员会任期5年，连续不得超过2届

                                          县级以上地方人民代表大会选举并且有权罢免本级监察委员会主任、本级人民院长和本级人民检察院检察长

                            # 刑法的适用范围

                            属地管辖

                            属人管辖，按国籍管辖

                            保护管辖，在外国构成犯罪的行为，我国判刑最低3年以上的可以管辖

                            普遍管辖，灭绝种族、人口贩卖

                            # 犯罪的本质

                            犯罪客体：刑法所保护的社会关系

                            犯罪主体：犯罪责任年龄+犯罪责任能力

                            刑事责任年龄

                            x⩾75 故意犯罪可以从轻或减轻、过失犯罪应当从轻或者减轻

                            x⩾16，应当负刑事责任

                            12⩽x<14 犯故意杀人、故意伤害罪、致人死亡或者以特别残忍手段致人重伤或严重残疾、情节恶劣，经最高人民检察院核准追诉的，应当负形式刑事责任

                            14⩽x<16 犯故意杀人、故意伤害致人重伤或者死亡、强奸、贩卖毒品、放火、爆炸、投毒的应当负刑事责任

                            14⩽x<18 应当从轻或减轻处罚

                            x<16 不予刑事处罚的，责令监护人加以管教

                            # 刑罚

                            主刑：管制[1]、拘役、有期徒刑、无期徒刑、死刑

                            附加刑：罚金、剥夺政治权利、没收财产、驱逐出境

                            有期徒刑：6个月~15年

                            数罪并罚，总和刑期不满35年的，不超过20年；总和刑期超过35年的，不超过25年

                            # 累犯

                            # 一般累犯

                            前后罪都是故意犯罪

                            前后罪都是有期徒刑以上

                            后罪发生在前罪执行完毕或者赦免后的5年内

                            # 特殊累犯

                            危害国家安全犯罪、恐怖活动犯罪、黑社会性质组织犯罪的犯罪分子，在任何时候再犯上述一类罪的，按累犯论处

                            # 假释

                            有期徒刑，执行不少于二分之一

                            无期徒刑实际执行13年以上的

                            不得假释的情形

                            犯暴力性犯罪（故意杀人、爆炸、抢劫、强奸、绑架、放火、投毒），被判处10年以上的，不得假释

                            1. 管制不关押人，但限制一定自由 ↩︎

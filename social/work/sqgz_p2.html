                                      # 社区建设

                                      # 社区建设的含义和意义

                                      名词解释

                                      社区建设是在党和政府的领导下，通过社区服务的方式，依靠社区的力量、利用社区的资源、强化社区功能，解决社区问题，以促进社区经济政治文化环境的协调发展，不断提高社区成员的生活水平和生活质量过程。

                                      # 社区建设的意义

                                      推进社区建设，是改革开放和社会主义现代化建设的迫切要求

                                      推进社区建设，是繁荣社区基层文化生活，加强社会主义精神文明建设的有效措施

                                      推进社区建设，是巩固城市基层政权和加强社会主义民主政治建设的重要途径

                                      # 社区建设的主要内容

                                      # 社区组织

                                      主要包括党建、社区自治组织建设、社区中介组织建设等

                                      # 社区服务

                                      • 老弱病残的社会救助
                                      • 面向社区居民的便民利民服务
                                      • 面向下岗职工的再就业服务

                                      # 社区卫生

                                      主要包括社区的疾病预防、医疗康复、心理咨询和计划生育

                                      # 社区文化

                                      主要包括社区群众性文化、教育、科普、体育娱乐等

                                      # 社区环境

                                      主要包括环境卫生、环境保护、绿化美化

                                      # 社区治安

                                      主要包括社区安全保卫、民事调解、群防群治、法制教育、法律咨询、流动人口管理和社会治安综合治理工作。

                                      # 社区建设的特点和功能

                                      # 社区建设的特点

                                      # 综合性

                                      社区建设是整个社区的全方位建设，不仅仅是特指某一方面的工作，而是对以往各种单 一的社区建设项目的整合。社区建设的综合性特征是由社区要素多样性和社区内容复杂性所决定的。

                                      # 社会性

                                      社区建设不是一种单纯的政府行为，也不是一种单纯的民间活动，而是各类社区主体、各种社区力量共同参与的过程。

                                      # 地域性

                                      社区是一种地域性的社会生活共同体，社区建设具有突出的地域性特征。

                                      # 群众性

                                      社区建设的基本原则是“以人为本”，社区居民的根本利益和实际需求是社区建设的出发点和立足点。

                                      # 多样性

                                      我国的社区建设没有统一模式，即使是同一地区或城市的不同社区，因经济发展水平不同、居民构成不同而使社区建设呈现出多样性。

                                      # 社区建设的功能

                                      # 管理协调功能

                                      在党的领导下，通过基层党组织和自治组织活动，发挥社区各类组织的作用，贯彻党的路线、方针、政策和加强社区思想政治工作，反映社情民意。

                                      # 社区服务功能

                                      包括面向老年人、残疾人、优抚对象、贫困户等社会特殊群体的社会福利服务，面向全体社区居民的便民利民服务和面向辖区单位的社会化服务网络。

                                      # 社会治安功能（综合服务）

                                      包括社区内的法制宣传、治安保安、民事调解、帮教失足青少年、流动人口管理、防火 防盗和其他社会治安综合管理工作

                                      # 医疗保健计生服务功能

                                      包括社区内成员的康复、医疗、保健和计划生育等需求

                                      # 环境维护功能

                                      包括社区的公共卫生、绿化、环境保护、市政设施维护等。

                                      # 文体教育功能

                                      包括各种群众性的文娱、体育、教育、科普活动以及多种形式的社会主义精神文明创建 活动。

                                      # 社会保障功能

                                      包括面向社区退休、失业人员和低收入家庭等社会困难群体的社会就业和保障服务工作。

                                      # 社区建设的指导思想、基本原则

                                      # 社区建设的指导思想

                                      邓小平理论、”三个代表“重要思想、科学发展观、习近平新时代中国特色社会主义思想为指导

                                      管理有序、服务完善、文明祥和的社会共同体作为基本目标

                                      # 社区建设的基本原则常考

                                      # 以人为本，服务居民

                                      社区建设的首要原则。坚持以不断满足社区居民的社会需求，提高居民生活质量和文明程度为宗旨，把服务社区居民作为社区建设的根本出发点和归宿。

                                      # 资源共享，共驻共建

                                      整合社区资源，充分调动社区内机关、团体、部队、企业事业组织等一切力量广泛参与社区建设，最大限度地实现社区资源的共有、共享，营造共驻社区、共建社区的良好氛围。

                                      # 责权统一，管理有序

                                      建立健全社区组织，明确社区组织的职责和权利，改进社区的管理与服务，寓管理于服务之中，逐步增强社区凝聚力。

                                      # 扩大民主，居民自治

                                      坚持按地域性、认同感等社区构成要素科学合理地划分社区；在社区内实行民主选举、民主决策、民主管理、民主监督，逐步实现社区居民自我管理、自我教育、自我服务、自我监督。

                                      # 因地制宜，循序渐进

                                      坚持实事求是，一切从实际出发，突出地方特色，从居民群众迫切要求解决和热切关注的问题入手，有计划、有步骤地实现社区建设的发展目标。
